# Activité Version de Workfront Fusion : semainedu mardi 23 mars 2026

Cette page décrit toutes les améliorations apportées à Adobe Workfront Fusion au cours de la semaine du mardi 23 mars 2026.

Pour obtenir la liste de toutes les modifications récentes, voir [Activité de publication d’Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Pour obtenir la liste des correctifs récents dans Workfront Fusion, reportez-vous à la page [Mises à jour de maintenance Workfront](https://experienceleague.adobe.com/fr/docs/workfront-known-issues/releases/current-updates) et recherchez toutes les mises à jour intitulées Mise à jour de maintenance de Workfront Fusion.

## Rendez un module d’appel API personnalisé (interrogation) désormais disponible dans le connecteur Adobe Authenticator

Pour qu’il soit possible de répéter des appels API jusqu’à ce qu’une certaine condition ou limite soit remplie, nous avons créé le module Effectuer un appel API personnalisé (interrogation) . Ce module comprend des options qui vous permettent de configurer des appels API répétés. Vous pouvez laisser l’appel API s’exécuter jusqu’à ce qu’une condition particulière soit remplie, définir un nombre spécifique de reprises et configurer le temps entre les reprises.

Pour plus d’informations, consultez la section [Effectuer un appel API personnalisé (interrogation)](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-authenticator-modules.md#make-a-custom-api-call-polling) dans l’article Modules Adobe Authenticator.
