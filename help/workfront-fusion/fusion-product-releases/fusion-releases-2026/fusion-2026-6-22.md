# Activité de publication de Workfront Fusion : semaine du 15 juin 2026

Cette page décrit toutes les améliorations apportées à Adobe Workfront Fusion au cours de la semaine du 15 juin 2026.

Pour obtenir la liste de toutes les modifications récentes, voir [Activité de publication d’Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Pour obtenir la liste des correctifs récents dans Workfront Fusion, reportez-vous à la page [Mises à jour de maintenance Workfront](https://experienceleague.adobe.com/fr/docs/workfront-known-issues/releases/current-updates) et recherchez toutes les mises à jour intitulées Mise à jour de maintenance de Workfront Fusion.

## Création de packages JavaScript personnalisés à utiliser dans les scénarios

Pour offrir une meilleure flexibilité et un meilleur contrôle de vos scénarios, nous avons ajouté la possibilité de créer des packages JavaScript personnalisés que vous pouvez ensuite utiliser dans les scénarios. Vous pouvez créer des packages personnalisés dans la zone Packages de Fusion. Vous ajoutez ensuite des fonctions ou des variables de ces packages à vos scénarios sous la forme d’un module Adobe App Builder.

Les packages incluent des fonctions, ainsi que toute variable ou dépendance sur laquelle ces fonctions s’appuient. Vous pouvez également tester des fonctions dans votre package avant de les utiliser dans vos scénarios

Comme les packages personnalisés fonctionnent via Adobe App Builder, votre entreprise doit disposer d’une licence Adobe App Builder pour les utiliser.

Pour plus d’informations sur l’utilisation de fonctions personnalisées dans Fusion, voir [ Utilisation de packages de fonctions personnalisées ](/help/workfront-fusion/create-scenarios/map-data/use-custom-function-packages.md).

## Afficher les modifications entre les versions de scénario

Pour faciliter la compréhension des modifications entre les versions de scénario, nous avons ajouté la possibilité d’afficher et de comparer ces modifications. Vous pouvez maintenant afficher une fenêtre qui affiche les modifications spécifiques entre deux versions sélectionnées du même scénario.

Pour plus d’informations, voir [Affichage et gestion des versions de scénario](/help/workfront-fusion/manage-scenarios/restore-a-scenario-version.md).
