# Activité Version de Workfront Fusion : semaine du mardi 9 février 2026

Cette page décrit toutes les améliorations apportées à Adobe Workfront Fusion au cours de la semaine du mardi 9 février 2026.

Pour obtenir la liste de toutes les modifications récentes, voir [Activité de publication d’Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Pour obtenir la liste des correctifs récents dans Workfront Fusion, reportez-vous à la page [Mises à jour de maintenance Workfront](https://experienceleague.adobe.com/fr/docs/workfront-known-issues/releases/current-updates) et recherchez toutes les mises à jour intitulées Mise à jour de maintenance de Workfront Fusion.

## Création de fonctions JavaScript personnalisées à utiliser dans les scénarios

Pour offrir une meilleure flexibilité et un meilleur contrôle de vos scénarios, nous avons ajouté la possibilité de créer une bibliothèque de fonctions JavaScript personnalisées que vous pouvez ensuite utiliser dans les scénarios. Vous pouvez créer des fonctions personnalisées dans la zone Fonctions de Fusion. Vous ajoutez ensuite ces fonctions à vos scénarios sous la forme d’un module Adobe App Builder .

Les fonctions personnalisées fonctionnent via Adobe App Builder. Votre entreprise doit donc disposer d’une licence Adobe App Builder pour les utiliser.

Pour plus d’informations sur l’utilisation de fonctions personnalisées dans Fusion, voir [Mappage de données à l’aide de fonctions personnalisées](/help/workfront-fusion/create-scenarios/map-data/map-using-custom-functions.md).

## Mettre à jour vers les mécanismes de sécurisation d’exécution incomplets

Pour mieux prendre en charge les scénarios complexes et volumineux, nous avons mis à jour les mécanismes de sécurisation pour les exécutions incomplètes. Désormais, les limites d’exécution incomplètes sont par scénario. Fusion peut stocker jusqu’à 1 Go ou 100 exécutions incomplètes par scénario, quelle que soit la limite atteinte en premier.

Auparavant, les limites étaient basées sur les équipes.

Pour plus d’informations sur les mécanismes de sécurisation dans Fusion, consultez [&#x200B; Mécanismes de sécurisation des performances de Fusion &#x200B;](/help/workfront-fusion/references/scenarios/fusion-performance-guardrails.md).

## Nouveau SharePoint Online Microsoft > Création d’un module de fichiers

Un nouveau module Créer un fichier est ajouté au connecteur SharePoint Online Microsoft. Le nouveau module offre de meilleures performances lorsque vous travaillez avec des fichiers volumineux.

Le module Créer un fichier , précédemment disponible, a été renommé Créer un fichier (hérité). Bien que le module hérité soit toujours disponible, nous vous recommandons d’utiliser le nouveau module.

Pour plus d’informations sur les modules Microsoft SharePoint Online, voir [Modules Microsoft SharePoint Online](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/sharepoint-modules.md).
