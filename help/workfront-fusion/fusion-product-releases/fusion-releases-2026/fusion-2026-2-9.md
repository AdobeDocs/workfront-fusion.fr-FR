# Activité Version de Workfront Fusion : semaine du mardi 9 février 2026

Cette page décrit toutes les améliorations apportées à Adobe Workfront Fusion au cours de la semaine du mardi 9 février 2026.

Pour obtenir la liste de toutes les modifications récentes, voir [Activité de publication d’Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Pour obtenir la liste des correctifs récents dans Workfront Fusion, reportez-vous à la page [Mises à jour de maintenance Workfront](https://experienceleague.adobe.com/fr/docs/workfront-known-issues/releases/current-updates) et recherchez toutes les mises à jour intitulées Mise à jour de maintenance de Workfront Fusion.

## Nouveau SharePoint Online Microsoft > Création d’un module de fichiers

Un nouveau module Créer un fichier est ajouté au connecteur SharePoint Online Microsoft. Le nouveau module offre de meilleures performances lorsque vous travaillez avec des fichiers volumineux.

Le module Créer un fichier , précédemment disponible, a été renommé Créer un fichier (hérité). Bien que le module hérité soit toujours disponible, nous vous recommandons d’utiliser le nouveau module.

Pour plus d’informations sur les modules Microsoft SharePoint Online, voir [Modules Microsoft SharePoint Online](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/sharepoint-modules.md).
