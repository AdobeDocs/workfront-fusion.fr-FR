# Activité Version de Workfront Fusion : semaine du mardi 16 février 2026

Cette page décrit toutes les améliorations apportées à Adobe Workfront Fusion au cours de la semaine du mardi 16 février 2026.

Pour obtenir la liste de toutes les modifications récentes, voir [Activité de publication d’Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Pour obtenir la liste des correctifs récents dans Workfront Fusion, reportez-vous à la page [Mises à jour de maintenance Workfront](https://experienceleague.adobe.com/fr/docs/workfront-known-issues/releases/current-updates) et recherchez toutes les mises à jour intitulées Mise à jour de maintenance de Workfront Fusion.

## Ajouter des invites d&#39;IA à vos scénarios avec le nouveau module Agent MCP

Vous pouvez inclure une invite d’IA dans votre scénario à l’aide du protocole MCP (Model Context Protocol) associé à des modèles de langage (LLM) volumineux. En les configurant dans le module Agent MCP, vous pouvez utiliser l’intelligence artificielle pour configurer des workflows efficaces, sécurisés et flexibles.

MCP vous permet d’utiliser une invite d’IA pour vous connecter aux applications. L’invite pouvant tenir compte des variations et du contexte, il n’est pas nécessaire de configurer manuellement certains workflows, itinéraires ou sections de scénario dans Fusion.

Pour plus d’informations sur l’utilisation des invites d’IA dans Fusion, voir [Ajouter une invite d’IA à votre scénario](/help/workfront-fusion/create-scenarios/add-modules/add-an-ai-prompt-to-your-scenario.md).

Pour plus d’informations sur la configuration du module de l’agent MCP, voir [Module de l’agent MCP](/help/workfront-fusion/references/apps-and-modules/tools-and-transformers/model-context-protocol-mcp-connector.md).

## API des opérations désormais disponible

Nous avons rendu possible l’obtention de données d’exploitation par le biais de l’API Workfront Fusion. Désormais, vous pouvez récupérer des données d’opérations détaillées et le nombre d’opérations pour leurs organisations, ainsi que regrouper et filtrer les résultats selon divers paramètres.

Pour plus d’informations, voir [API Workfront Fusion](https://developer.adobe.com/workfront-fusion-apis/#)

## Affichage des mesures de performances à l’échelle de l’organisation dans le tableau de bord des performances de Fusion

Pour faciliter la compréhension des performances de Fusion au sein de votre organisation, nous avons créé le tableau de bord des performances . Désormais, les administrateurs et administratrices de Fusion peuvent rapidement identifier les scénarios les plus exécutés, les endroits où les retards se produisent et l’efficacité du fonctionnement des pools de programmes de travail. Le tableau de bord des performances offre une visibilité en temps réel sur les volumes d’exécution, la profondeur de file d’attente, l’utilisation du pool et les performances au niveau du scénario.

Pour plus d’informations sur le tableau de bord des performances de Fusion, voir [Afficher le tableau de bord des performances d’une organisation](/help/workfront-fusion/set-up-and-manage-workfront-fusion/set-up-and-manage-orgs-and-teams/set-up-orgs-teams-and-users/view-performance-dashboard.md).
