# Activité de publication de Workfront Fusion : semaine du 2 mars 2026

Cette page décrit toutes les améliorations apportées à Adobe Workfront Fusion au cours de la semaine du 2 mars 2026.

Pour obtenir la liste de toutes les modifications récentes, voir [Activité de publication d’Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Pour obtenir la liste des correctifs récents dans Workfront Fusion, reportez-vous à la page [Mises à jour de maintenance Workfront](https://experienceleague.adobe.com/fr/docs/workfront-known-issues/releases/current-updates) et recherchez toutes les mises à jour intitulées Mise à jour de maintenance de Workfront Fusion.

## Création de fonctions JavaScript personnalisées à utiliser dans les scénarios

Pour offrir une meilleure flexibilité et un meilleur contrôle de vos scénarios, nous avons ajouté la possibilité de créer une bibliothèque de fonctions JavaScript personnalisées que vous pouvez ensuite utiliser dans les scénarios. Vous pouvez créer des fonctions personnalisées dans la zone Fonctions de Fusion. Vous ajoutez ensuite ces fonctions à vos scénarios sous la forme d’un module Adobe App Builder .

Les fonctions personnalisées fonctionnent via Adobe App Builder. Votre entreprise doit donc disposer d’une licence Adobe App Builder pour les utiliser.

Pour plus d’informations sur l’utilisation de fonctions personnalisées dans Fusion, voir [Mappage de données à l’aide de fonctions personnalisées](/help/workfront-fusion/create-scenarios/map-data/map-using-custom-functions.md).

## Autorisation de jeton d’accès personnel désormais disponible pour le connecteur Airtable

Pour rendre le connecteur Airtable plus flexible, nous avons ajouté la possibilité d’utiliser des jetons d’accès personnel (PAT) pour les autorisations. Désormais, lors de la création d’une connexion, vous pouvez choisir d’utiliser l’autorisation OAuth2 ou PAT.

Pour obtenir des informations et des instructions, voir [Modules Airtable](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/airtable-modules.md).
