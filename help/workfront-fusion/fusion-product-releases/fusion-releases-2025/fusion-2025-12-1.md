# Activité Version de Workfront Fusion : semaine du mercredi 2 décembre 2025

Cette page décrit toutes les améliorations apportées à Adobe Workfront Fusion au cours de la semaine du mercredi 2 décembre 2025.

Pour obtenir la liste de toutes les modifications récentes, voir [Activité de publication d’Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Pour obtenir la liste des correctifs récents dans Workfront Fusion, reportez-vous à la page [Mises à jour de maintenance Workfront](https://experienceleague.adobe.com/fr/docs/workfront-known-issues/releases/current-updates) et recherchez toutes les mises à jour intitulées Mise à jour de maintenance de Workfront Fusion.

## Nouvelles variables disponibles dans le panneau de mappage

Nous avons ajouté les nouvelles variables suivantes à la zone Fonctions générales du panneau de mappage :

* `scenarioID` : ID du scénario actuellement ouvert.
* `operationsConsumed` : nombre d’opérations utilisées à ce stade du scénario.

Auparavant, ces informations n’étaient pas disponibles pour le mappage.

Pour plus d’informations sur le mappage, voir [Présentation du mappage](/help/workfront-fusion/get-started-with-fusion/understand-fusion/mapping-overview.md).

Pour plus d’informations sur le mappage, consultez les articles sous [Mapper des données : index d’article](/help/workfront-fusion/create-scenarios/map-data/map-data-toc.md).

## Opérations consommées désormais disponibles dans le panneau de mappage

## Affichez les abonnements aux événements associés à un webhook Workfront.

Pour faciliter l’affichage des informations sur un Webhook Workfront, nous avons ajouté des informations d’abonnement aux événements. Désormais, vous pouvez afficher des informations sur les abonnements aux événements Workfront associés à un webhook Workfront.

Ces informations sont visibles dans le module Workfront > Événements Espion , ainsi que dans la zone Webhooks de Fusion.

Pour plus d’informations sur le module Workfront > Événements Espion, voir [Événements Espion](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/workfront-modules.md#triggers) dans l’article Modules Workfront.

Pour plus d’informations sur la zone des Webhooks, voir [Déclencheurs instantanés (webhooks)](/help/workfront-fusion/references/modules/webhooks-reference.md).

## Recherche de modules dans un scénario

Pour faciliter la localisation d’un module spécifique dans un scénario, nous avons ajouté la possibilité de rechercher l’éditeur de scénarios. Désormais, vous pouvez rechercher des modules par identifiant, nom, application ou type de module. Vous pouvez ensuite cliquer sur les résultats de la recherche pour accéder directement à un module spécifique.

Pour plus d’informations sur l’éditeur de scénarios, voir [L’éditeur de scénarios](/help/workfront-fusion/get-started-with-fusion/navigate-fusion/scenario-editor.md).
