# Modules [!UICONTROL Draft.js]

Dans un scénario Adobe Workfront Fusion, vous pouvez utiliser les modules [!UICONTROL Draft.js] pour convertir Draft.js en HTML et HTML en Draft.js. Vous pouvez l’utiliser conjointement avec les modules Workfront pour ajouter du texte enrichi ou lire du texte enrichi à partir des champs Workfront.

Pour obtenir des instructions sur la création d’un scénario, consultez les articles sous [Créer des scénarios : index des articles](/help/workfront-fusion/create-scenarios/create-scenarios-toc.md).

Pour plus d’informations sur les modules, consultez les articles sous [Modules : index des articles](/help/workfront-fusion/references/modules/modules-toc.md).
